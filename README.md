# VVVF-Invertor-Controller
I made a custom VVVF Invertor Controller based on STM32 microcontroller .It uses 2 voltage relulators 3.3V (MCU, logic) 5V (interfaces, sensors), it has a Gate Driver Interface and Communication Interfaces.

## Features
- 3-phase inverter control <br>
- Supports PWM
- STM32 microcontroller based
- 6 independent PWM outputs
- Phase current sensing inputs
- DC bus voltage monitoring
